# Gmail Dictionary Attack.

This is a gmail dictionary attack script
written using pure bash. 

- Installation

    $ git clone https://github.com/d4az/gmail-hack.git


# How to Run the Script ? 

    $./hack-gmail.sh --help
         

# To perform dictionary attack ?

    $./hack-gmail.sh --gmail
